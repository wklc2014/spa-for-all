## 当前时间
`var now = moment();`

## 字符串

```
moment(String);

```

## 字符串 + 格式

```
moment(TimeString, FormatString);
moment(TimeString, FormatString, String);
moment(TimeString, FormatString, Boolean);
moment(TimeString, FormatString, String, Boolean);
```
